# 🪙 Vit Crypto Price

Uma aplicação web moderna e elegante para monitorar os preços das principais criptomoedas em tempo real.

![Preview](assets/caricatura.png)

## 📋 Descrição

**Vit Crypto Price** é um dashboard de preços de criptomoedas que exibe informações atualizadas em tempo real. A aplicação consome dados da API CoinGecko e apresenta uma interface intuitiva com indicadores visuais de tendências de preço.

## ✨ Funcionalidades

- 📊 **Preços em Tempo Real** - Dados atualizados da API CoinGecko
- 💹 **Variação 24h** - Mostra a mudança de preço nas últimas 24 horas
- 🎨 **Interface Visual** - Design moderno com cores:
  - 🟢 Verde para criptomoedas em alta
  - 🔴 Vermelho para criptomoedas em queda
- ⚡ **Rápido e Responsivo** - Carregamento instantâneo de dados
- 📱 **Design Responsivo** - Funciona em desktop e mobile

## 💰 Criptomoedas Monitoradas

- Bitcoin (BTC)
- Ethereum (ETH)
- Tether (USDT)
- Litecoin (LTC)
- Cardano (ADA)
- Dogecoin (DOGE)

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/Vit-Crypto-Price.git
cd Vit-Crypto-Price
```

2. Abra o arquivo `index.html` no seu navegador:
   - Duplo clique em `index.html`, ou
   - Use um servidor local para melhor experiência

3. A aplicação carregará automaticamente e exibirá os preços das criptomoedas em tempo real

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização moderna com fonts do Google
- **JavaScript (Vanilla)** - Lógica sem frameworks
- **CoinGecko API** - Dados de preços em tempo real

## 📖 API Utilizada

Este projeto utiliza a [CoinGecko API](https://www.coingecko.com/api/documentation) para obter dados de preços de criptomoedas em tempo real. A API é:
- ✅ Gratuita
- ✅ Sem autenticação necessária
- ✅ Dados atualizados em tempo real

## 🎨 Customização

Você pode facilmente:
- Adicionar novas criptomoedas editando a URL da API em `script.js`
- Modificar as cores e estilo em `style.css`
- Alterar o design da interface conforme desejar
